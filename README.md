# Sushi Store

A premium Sushi E-commerce platform built with Laravel, Livewire, and Midtrans Integration.

## Features
- **Order Tracking**: Real-time order progress for customers.
- **Admin Panel**: Responsive dashboard for managing products, categories, and orders.
- **Midtrans Integration**: Secure payment gateway for seamless transactions.
- **User Management**: Track and manage registered customers.

## Technologies Used
- Laravel
- Livewire
- Alpine.js
- Vanilla CSS (Premium Designs)
- Midtrans
